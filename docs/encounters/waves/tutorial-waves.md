## 1. 教學目標

### 1.1 學習順序

1. **移動與攻擊** — 熟悉英雄操作
2. **部署幹員** — 理解塔防機制
3. **技能使用** — 掌握位移技
4. **收集資源** — 理解 Cost 系統
5. **終極技能** — 體驗大招

## 2. 波次設計

### Wave 1: 基礎移動

| 項目 | 內容 |
|------|------|

| 難度 | 極易 |
| 目標 | 擊殺所有敵人 |

| 項目 | 內容 |

| 項目 | 內容 |
