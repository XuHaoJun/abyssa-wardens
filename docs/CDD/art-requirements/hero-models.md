# 英雄 3D 模型规格 (Hero Models)

## 🗣️ 內部討論紀錄 (Internal Discussion Record)

**任務：** Phase 1 - Task 5：設計英雄 3D 模型規格

**與會者：**
- 🔥 **Diablo 代表** — 敘事深度、黑暗氛圍、裝備驅動
- ⚡ **PoE 代表** — 系統複雜度、build 可能性
- 🛡️ **Arknights 代表** — 策略性塔位、角色魅力
- 🏰 **Kingdom Rush 代表** — 塔防核心樂趣、平衡性

---

**討論摘要：**

**Diablo 代表：** 「英雄是玩家操作的焦点，需要有足够的辨识度和表现力。」

**PoE 代表：** 「模型需要有足够的细节来展示装备外观，同时保持风格统一。」

**Arknights 代表：** 「角色需要有独特的轮廓，让玩家一眼就能认出是谁。」

**Kingdom Rush 代表：** 「TD 视角下，角色需要有足够的高度和存在感。」

---

**共識結論：**
- **风格：** Dark Sci-Fantasy
- **视角：** 俯视 2.5D，需要有足够的厚度感
- **表现：** 强调剪影轮廓和配色区分

---

## 1. 機甲軍工師 (Ironclad Engineer)

### 1.1 角色概述

| 项目 | 内容 |
|------|------|
| 高度 | 约 2.2 单位高度 |
| 体型 | 厚重装甲包裹 |
| 配色 | 深灰 + 橙色光芒 |
| 武器 | 背后携带折叠式工事平台 |

### 1.2 模型要求

| 组件 | 描述 | 优先级 |
|------|------|--------|
| 主体 | 厚重的外骨骼装甲 | P0 |
| 头部 | 战术目镜 + 呼吸器 | P0 |
| 背部 | 折叠式工事平台（可展开） | P1 |
| 肩部 | 武器挂点 + 推进器 | P1 |
| 腿部 | 液压关节 + 稳定支架 | P2 |

### 1.3 材质要求

- **装甲：** 金属质感 + 磨损痕迹
- **关节：** 发光液压管线（橙色）
- **目镜：** 发光效果（橙色常亮，技能时闪烁）

### 1.4 动画需求

| 动画类型 | 时长 | 备注 |
|----------|------|------|
| 待机 | 3秒循环 | 轻微的液压运作动画 |
| 移动 | 1秒循环 | 沉重的金属脚步 |
| 攻击 | 0.5秒 | 机枪扫射动作 |
| 位移 | 0.8秒 | 推进跳跃 |
| 受击 | 0.3秒 | 装甲碰撞效果 |
| 终极技能 | 2秒 | 全身发光 + 能量波 |

## 3. 通用规格

### 3.1 渲染风格

- **整体风格：** PBR 渲染 + 卡通渲染混合
- **光影：** 强调轮廓光，确保俯视视角清晰
- **特效：** 技能特效需要醒目但不遮挡视野

### 3.2 性能要求

- **多边形数：** 每英雄不超过 15,000 面
- **骨骼数：** 每英雄不超过 80 根骨骼
- **材质数：** 每英雄不超过 8 个材质

### 3.3 适配要求

- 需要适配键鼠和手柄两种操作模式
- 视角：俯视 45-60 度
