**討論摘要：**

**Diablo 代表：** 「12 种干员要覆盖所有基础玩法——近战阻挡、远程输出、辅助增益。」

**PoE 代表：** 「每个干员是『塔基座』——属性差异来自阻擋數、HP、Range，而不是自带技能。攻击模式完全由装备上的宝石决定。」

**Arknights 代表：** 「在 AK 里，干员分职业——先锋、近卫、狙击、辅助、重装。我们的分类可以参考这个逻辑。」

**Kingdom Rush 代表：** 「TD 的核心是兵种相克。干员的价值在于他们的基础属性差异——重装高血量阻挡，狙击远距离输出。」

---

**共識結論：**

- **幹員 = 陣地石 (塔基座)** — 不是完整的战斗单位
- **攻击方式** — 完全由连接的🔴技能石 + 🔵辅助石决定
- **5 大分类：** 重装 (Tank)、近卫 (Melee DPS)、狙击 (Ranged DPS)、术士 (Caster)、辅助 (Support)
- **12 种干员** — 每分类 2~3 种，差异在于基础属性

## 1. 幹員分類體系

### 1.1 五大职业分类

| 分类 | 缩写 | 部署位置 | 核心功能 |
|------|------|----------|----------|
| 重装 | TNK | 平地 | 阻挡怪物、吸收伤害 |
| 近卫 | MEL | 平地 | 中距离输出、阻挡精英怪 |
| 狙击 | RNG | 高台 | 超远距离单体输出 |
| 术士 | MAG | 高台 | 远程 AOE 输出 |
| 辅助 | SUP | 高台/平地 | 增益、治疗、减速 |

### 1.2 部署位置规则

- **平地 (Ground)：** 只能部署「重装」和「近卫」
- **高台 (Highground)：** 只能部署「狙击」「术士」和部分「辅助」
- **例外：** 英雄技能可以改变局部地形（如机甲方军工师的临时高台）

## 4. 幹員與寶石的互動（重要說明）

**幹員本身不攜帶技能** — 攻擊模式完全由裝備上的🔴技能石決定。

**範例：**
- 將 🔴[旋風斬技能石] + 🔵[範圍擴大] 連線在 🔵[劍舞者] 上
- 部署劍舞者 → 自動釋放旋風斬 + 範圍擴大

**過載機制 (Overload)：**
- 英雄累積「統帥值」可手動觸發
- 效果：加速已部署幹員的技能觸發頻率（10秒內）
- 這是玩家唯一的「手動干預幹員」手段
