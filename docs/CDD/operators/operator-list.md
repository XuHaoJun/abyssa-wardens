# 幹員清單與分類 (Operator List)

## 🗣️ 內部討論紀錄 (Internal Discussion Record)

**任務：** Phase 1 - Task 3：設計 12 種幹員清單與分類

**與會者：**
- 🔥 **Diablo 代表** — 敘事深度、黑暗氛圍、裝備驅動
- ⚡ **PoE 代表** — 系統複雜度、build 可能性
- 🛡️ **Arknights 代表** — 策略性塔位、角色魅力
- 🏰 **Kingdom Rush 代表** — 塔防核心樂趣、平衡性

---

**討論摘要：**

**Diablo 代表：** 「12 种干员要覆盖所有基础玩法——近战阻挡、远程输出、辅助增益、特殊机制。」

**PoE 代表：** 「每个干员都要有独特的定位，不能只是伤害高低之分。最好每种都有 1-2 个能影响战局的特殊技能。」

**Arknights 代表：** 「在 AK 里，干员分职业——先锋、近卫、狙击、辅助、重装、特种。我们的分类可以参考这个逻辑，但要做简化。」

**Kingdom Rush 代表：** 「TD 的核心是兵种相克。12 种干员要形成互相制约的关系——比如重装克高攻怪、刺客克闪避怪。」

---

**共識結論：**
- **5 大分类：** 重装 (Tank)、近卫 (Melee DPS)、狙击 (Ranged DPS)、术士 (Caster)、辅助 (Support)
- **12 种干员：** 每分类 2~3 种，覆盖不同玩法
- **核心原则：** 每个干员都有独特的「主动技能」，不只是被动属性差异

---

## 1. 幹員分類體系

### 1.1 五大职业分类

| 分类 | 缩写 | 部署位置 | 核心功能 |
|------|------|----------|----------|
| 重装 | TNK | 平地 | 阻挡怪物、吸收伤害 |
| 近卫 | MEL | 平地 | 高输出、阻挡精英怪 |
| 狙击 | RNG | 高台 | 远程单体输出 |
| 术士 | MAG | 高台 | 远程 AOE 输出 |
| 辅助 | SUP | 高台/平地 | 增益、治疗、控场 |

### 1.2 部署位置规则

- **平地 (Ground)：** 只能部署「重装」和「近卫」
- **高台 (Highground)：** 只能部署「狙击」「术士」和部分「辅助」
- **例外：** 英雄技能可以改变局部地形（如机甲方军工师的临时高台）

## 3. 幹員屬性模板

每种干员拥有以下基础属性（数值会在 JSON 中详细定义）：

| 属性 | 说明 |
|------|------|
| HP | 生命值上限 |
| ATK | 攻击力 |
| DEF | 物理防御 |

| Range | 攻击范围 |
| Block | 阻挡数 |
| Deploy Cost | 部署费用 |
| ReDeploy Time | 再部署冷却 |

## 4. 独特机制说明

每个干员都有一个独特的「主动技能」，在战斗中可以手动触发：

| 干员 | 主动技能名称 | 效果 |
|------|-------------|------|
| 盾衛 | 盾击 | 晕眩前方敌人 2 秒 |
| 鐵壁 | 战吼 | 范围内友军 +30% 防御，持续 10 秒 |
| 荊棘 | 尖刺领域 | 周围敌人攻速 -50%，持续 5 秒 |
| 劍舞者 | 旋风斩 | 旋转攻击周围所有敌人 |
| 戰斧 | 致命打击 | 下一次攻击伤害 x3 |
| 狙擊手 | 瞄准 | 下一次攻击必定暴击，且距离无限 |
| 弓騎士 | 穿透箭 | 攻击穿透一条直线上的所有敌人 |
| 元素師 | 元素切换 | 在冰/火/雷之间切换属性 |
| 死靈法師 | 召唤骷髅 | 召唤 2 个骷髅战士协助作战 |
| 扭曲師 | 置换 | 将自身与范围内最远的敌人位置互换 |
| 光牧 | 治疗波 | 恢复范围内所有友军 30% 生命 |
| 减速塔 | 冰霜领域 | 冻结区域内所有敌人 1 秒 |
