## 2. 部署點類型

### 2.1 平地 (Ground)

| 屬性 | 數值 |
|------|------|
| 可部署職業 | 重裝 (TNK)、近衛 (MEL)、輔助 (SUP) |
| 阻擋能力 | 有 |
| 佔用面積 | 1 格 |

### 2.2 高台 (Highground)

| 屬性 | 數值 |
|------|------|
| 可部署職業 | 狙擊 (RNG)、術士 (MAG)、輔助 (SUP) |
| 阻擋能力 | 無 |
| 佔用面積 | 1 格 |
| 高度差 | +3 格 |

### 2.3 臨時高台 (Temporary Highground)

| 屬性 | 數值 |
|------|------|
| 來源 | 英雄技能（機甲軍工師） |
| 持續時間 | 20 秒 |
| 可部署職業 | 遠程職業 |
| 數量上限 | 2-3 座 |

---

## 3. 部署規則

### 3.1 費用消耗

- 部署消耗「部署點數 (Cost)」
- Cost 不足無法部署
- 撤退返回 50% Cost

### 3.2 冷卻時間

| 幹員稀有度 | 再部署冷卻 |
|------------|------------|
| 普通 (N) | 15-20 秒 |
| 精良 (R) | 18-25 秒 |
| 傳奇 (L) | 20-30 秒 |

### 3.3 空間限制

- 每格只能部署 1 個幹員
- 鄰近格可部署多個幹員
- 路徑上不可部署

## 4. 地形互動

### 4.1 可摧毀障礙物

| 類型 | 佔用格數 | 效果 |
|------|----------|------|
| 藤蔓牆 | 1 | 阻擋敵人，可摧毀 |
| 木製障礙 | 1 | 阻擋敵人，可摧毀 |

### 4.2 環境效果

| 地形 | 對敵人效果 | 對幹員效果 |
|------|------------|------------|
| 沼澤 | 減速 30% | 無 |
| 水窪 | 無 | 無 |

1. 選擇部署位置

3. 確認職業符合位置
4. 確認無空間衝突
5. 扣除 Cost，部署完成
